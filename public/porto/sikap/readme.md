Website Presensi Untuk Dinas Pendidikan Kabupaten Wajo

website ini digunakan untuk melakukan presensi bagi para dinas dalam kabupaten wajo 

fitur yang ada pada website ini adalah:
- login
- Presensi Pegawai
- Pengajuan Pegawai
- laporan presensi
- Aktfitas Guru

dan masih banyak lainya

project ini mempunyai stack laravel sebagai API & React JS sebagai frontend dashboard
